]]> Wisen.tech is looking a talented Java Software Developer Contractor to design and code industry-disruptive products with the specified technology and the current project stage like analysis, software architecture/design,  development/configuration, test, release, run/support.

If you want to build amazing products and tackle complex questions and to be a part of international projects you’ll have the opportunity to just that.

The project location : Neu-Isenburg, Frankfurt-Germany

As a Software Engineer you will:
  • Work in a project in the context of a Microservice Architecture based on Java and Spring-Boot
  • Design and implement for resilient and effective solutions with their full lifecycle of design, code, testing, and maintenance
  • Develop of implementations based on SOAP and REST Micro-Services in Kafka and Hadoop
  • Learn, discover, evolve, and mentor continually
The Skills required:
  • Academic education in computer science or a comparable qualification and several years of project experience
  • Spring Framework / Spring Boot
  • JUnit, Selenium, Tosca, Git
  • XML/XSD, Json
  • Docker
  • Kafka, Hadoop/Spark
  • Knowledge of event stream processing and event-based integration would be helpful
  • Practical knowledge of CI/CD tools especially Jenkins, Nexus, Ansible and Selenium, would be a benefit
  • A fluency in English is essential.

Duration : 6 months

To apply please send your resume to cv@wisen.tech
